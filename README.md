# Simulador educativo del funcionamiento de la web

Proyecto web estático que muestra, de forma visual y accesible, el recorrido que sigue una URL desde que el usuario la escribe hasta que el navegador renderiza la página. El sitio está organizado en tres páginas: Inicio, Contenido y Contacto.

## Framework utilizado
- Tailwind CSS: se empleó para crear un diseño responsive, con navegación adaptable, botones, cards, formularios, alertas y badges.

## Características principales
- Diseño responsive para móvil, tablet y escritorio
- Navegación clara y accesible
- Uso de etiquetas HTML5 semánticas ("header", "nav", "main", "section", "article", "footer")
- Contraste adecuado entre texto y fondo
- Texto alternativo en imágenes (`alt`)
- Estados visibles en enlaces y botones (`hover` y `focus`)

## Estructura del proyecto
- "index.html": página de inicio
- "contenido.html": contenido temático
- "contacto.html": recursos y formulario
- "css/styles.css": estilos personalizados
- "js/menu.js": menú responsive
- "img/": recursos visuales
